# 🧬 PLACEHOLDER: Gemelo Digital Soberano

Infraestructura crítica orientada a eliminar la fragmentación de la información de salud en Chile.

## 🚀 Objetivos Estratégicos
- **Interoperabilidad Universal:** Estándar HL7 FHIR.
- **Soberanía Digital:** El paciente es dueño de su dato.
- **Legacy Gateway:** Digitalización de documentos físicos mediante IA.

## 🛠️ Stack Tecnológico
- **Backend:** FastAPI.
- **Frontend:** Streamlit.
- **Base de Datos:** SQLite.
- **IA:** DeepSeek API.
